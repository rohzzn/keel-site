Data safety

What data may be used, and why.

Keel is designed to minimize noise and keep data handling understandable. This page summarizes the categories of data used by the app.

Last updated: March 7, 2026

Account data

If optional cloud sync is enabled, Keel may store the user's email address and account identifier so the same data can be restored on another device.

Subscription data

The app stores subscription names, prices, billing cycles, billing dates, payment method labels, notes, and related settings needed to provide the product experience.

Photos or media

If a user uploads a custom logo, Keel may access the photo library only for the image the user chooses.

Local and cloud behavior

Keel stores data locally by default. If cloud sync is enabled, synced app state may also be stored by the cloud sync provider to support restore and multi-device continuity.

Advertising

Keel does not currently use third-party ad networks and does not currently sell user data.
