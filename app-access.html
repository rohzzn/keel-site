Reviewer access

No login required for the core app.

Keel's main subscription tracking experience is available without signing in. Optional cloud sync can be enabled later, but it is not required to review or use the core product.

Last updated: March 7, 2026

What reviewers can access immediately

  • Add, edit, and delete subscriptions.
  • Browse the service catalog.
  • Use the billing calendar and spending overview.
  • Configure reminders, payment methods, and currencies.
  • Export subscription data.

Optional account flow

Users may create an optional account if they want cloud sync across devices. Cloud sync is not required for core tracking or review.
