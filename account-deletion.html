Account deletion

Delete optional cloud sync data.

Keel can be used without an account. If a user created an optional cloud sync account and wants the associated cloud data removed, follow the request steps on this page.

Last updated: March 7, 2026

What can be deleted

Deletion requests apply to the optional cloud sync account and associated synced subscription data stored for that account.

How to request deletion

  • Email rohanpothuru@gmail.com from the email address associated with your Keel cloud sync account.
  • Use the subject line: Account Deletion Request.
  • Include any detail needed to identify the account if you have changed your sign-in email.

Response time

Keel aims to process valid deletion requests within 7 business days.

Local device data

Local app data stored on a device can be cleared inside the app by using the reset option in Settings.
