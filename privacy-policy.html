Privacy policy

Privacy, written plainly.

Keel is designed to keep subscription management simple. This policy explains what information the app stores, when data may be sent to the optional cloud sync service, and how users can request support.

Last updated: March 7, 2026

What Keel stores

Keel stores the subscription information that users enter, such as service name, price, billing cycle, next billing date, payment method labels, notes, custom logo references, and optional URLs.

Local-first storage

By default, subscription data and settings are stored on the device so the app can work quickly offline and preserve the user's state between sessions.

Optional cloud sync

If a user enables cloud sync, Keel sends account and app-state data to its cloud sync provider so the same subscription data can be restored on another device signed into the same account. Cloud sync is optional and not required to use the core app experience.

Photos and custom logos

If a user chooses a custom subscription logo, Keel may request access to the device photo library so the selected image can be used inside the app. Keel does not scan the photo library beyond the image the user explicitly selects.

Notifications

If billing reminders are enabled, Keel schedules reminder notifications for upcoming billing dates. These reminders are controlled by the user inside the app.

Advertising and sale of data

Keel does not currently show third-party advertising and does not sell personal information.

Access, deletion, and support

Users can remove locally stored app data from within the app by using the reset option. For cloud sync account deletion or data deletion requests, contact contact@usekeel.co.

Policy updates

Keel may update this policy as the product evolves. Material changes will be reflected by updating the date on this page.

Need help?

Privacy questions, deletion requests, and reviewer questions can be sent to contact@usekeel.co.
